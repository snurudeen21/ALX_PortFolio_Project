Soil Report Lab is a Soil Classification Web App built using python programming language

and deployed on the web using the flask framework and other frontend technologies such as: HTML, CSS, and JavaScript

Users can upload parameters from a sieve analysis test like: sieve size, cumulative percent retained, and percentage

passing. These parameters will be retrieved from an excel file in csv format. Alternatively, users can enter these

parameters manually. Liquid limit and plastic limit can only be entered manually by the user.

The data entered will be used to classify soils based on AASHTO and USCS classification.

The app will display graph of the soil gradation curve based on the parameters provided

Also there will be an additional feature so save a soil report in PDF file format
